# Badatro - Simple Godot Balatro Clone

Implemented the basic gameplay loop of Balatro, with custom pixel art for 35 Jokers, all 32 vouchers and a 52 card deck.

![Dashboard Image (playing)](images/screenshots/playing.png)

There exists a few bugs for card animation, and boss blinds are yet to be implemented.
